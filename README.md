# SheepTrackAI

Web aplikacija za digitalno vođenje evidencije ovčarske farme.

## Funkcionalnosti

* Upravljanje OPG-om
* Evidencija grla
* Evidencija janjenja
* Evidencija janjadi
* Evidencija pripusta
* Veterinarski zapisi i cijepljenja
* Evidencija vaganja
* Arhiva grla
* Dashboard sa statistikom i grafovima
* JSON API za statistiku
* Uvoz grla iz e-Posjednika (PDF)

## Tehnologije

* Python
* Django 4.2
* MariaDB / MySQL
* Bootstrap 5
* JavaScript
* Chart.js
* Django REST Framework

## Instalacija

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
