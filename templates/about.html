O aplikaciji i literatura

SheepTrackAI Web

SheepTrackAI je web aplikacija za digitalno vođenje evidencije ovčarske farme. Aplikacija omogućuje upravljanje OPG-om, grlima, janjenjima, janjadi, pripustima, veterinarskim zapisima, vaganjima i arhivom grla.

Sustav koristi Django backend, lokalnu SQLite bazu podataka, Bootstrap dizajn i JavaScript za dinamičke dijelove korisničkog sučelja.

Korištene tehnologije

  • Python
  • Django
  • SQLite baza podataka
  • HTML
  • CSS
  • Bootstrap 5
  • JavaScript
  • AJAX / Fetch API
  • JSON
  • Git i GitHub

JSON API

Aplikacija sadrži vlastiti Django API endpoint koji vraća statistiku aplikacije u JSON formatu. Frontend dohvaća podatke pomoću JavaScript Fetch API-ja.
